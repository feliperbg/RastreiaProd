SMP - SALARIO MENSAL PROGRAMADOR
SDIAP = SMP / 30
SHORAP = SDIAP / 8

CUSTO SISTEMA PROGRAMADOR =  SHORAP * QTDHORASPROGRAMADOR

------------------------------------------

SMA - SALARIO MENSAL ANALISTA
SDIAA = SMA / 30
SHORAA = SDIAA / 8

CUSTO SISTEMA ANALISE =  SHORAA * QTDHORASANALISTA

-------------------------
CUSTO TOTAL = CUSTO SISTEMA ANALISE + CUSTO SISTEMA PROGRAMADOR
